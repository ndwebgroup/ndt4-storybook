# CSS Utility Classes

Available utility classes for layout, spacing, visibility, backgrounds, and more.

### Note

- Responsive classes (e.g., `.visually-hidden-md`, `.col--sm`) apply at specific breakpoints.
- All color and spacing variables (e.g., `--brand-blue`, `--gutter-width`) are defined as part of the base theme.

---

### Visibility

| Class                | Description                                                                 |
|----------------------|-----------------------------------------------------------------------------|
| `.hidden`            | Completely hides the element (`display: none`).                             |
| `.invisible`         | Makes the element invisible but it still takes up space (`visibility: hidden`). |
| `.visually-hidden`   | Hides visually but accessible to screen readers.                            |
| `.visually-hidden-md`, `.visually-hidden-ml`, `.visually-hidden-lg`, `.visually-hidden-xl`, `.visually-hidden-xxl` | Responsive visually hidden classes for different breakpoints. |
| `.filter-group:not(:has(li.filter-record:not(.hidden)))` | Hides filter group if no visible filter records. |
| `.wrap-link`         | Forces long links to wrap and break as needed.                              |

### Margin and Padding

Margin (`m-`) and padding (`p-`) utilities, including block (`mb-`, `pb-`) and inline (`mi-`, `pi-`) variants.

| Class Example | Description                |
|---------------|---------------------------|
| `.m-0`        | No margin                  |
| `.m-1`        | 0.5rem margin              |
| `.m-2`        | 1rem margin                |
| `.m-auto`     | Auto margin                |
| `.mi-gutter`  | Inline margin using gutter |
| `.p-0`        | No padding                 |
| `.p-1`        | 0.5rem padding             |
| `.pb-5`       | 4rem block padding         |
