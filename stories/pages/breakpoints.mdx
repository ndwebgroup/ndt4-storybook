    # Responsive Breakpoints

    The ND Web Theme has been developed to be mobile first, using media queries to create breakpoints for our layouts. Here are the primary breakpoints we use:
  </div>
  <div className="sb-section">
    ```css
      /* Extra small - xs (< 480px) */
      /* No media query. This is default. */

      /* Small - sm (>= 480px) */
      @media only screen and (width >= 30em) { ... }

      /* Medium - md (>= 768px) */
      @media only screen and (width >= 48em) { ... }

      /* Medium-Large - ml (>= 960px) */
      @media only screen and (width >= 60em) { ... }

      /* Large - lg (>= 1280px) */
      @media only screen and (width >= 80em) { ... }

      /* Extra large - xl (>= 1440px) */
      @media only screen and (width >= 90em) { ... }

      /* Extra-extra-large - xxl (>= 1600px) */
      @media only screen and (width >= 100em) { ... }
    ```
  </div>
