## CSS Custom Properties
The following CSS custom properties are used throughout the Web Theme to designate the appropriate font-stack:

```css
/* Default */
--font-default: GP, "Helvetica Neue", Helvetica, Arial, Verdana, sans-serif;

/* Headings */
--font-heading: Garamond-Pro, "Adobe Garamond", Garamond, Georgia, "Times New Roman", Times, serif;
```

## Usage

### Headings
- Brand Blue and Dark Gray are the only colors designated for headings when used within body copy. White may be used when the heading is displayed over a dark background. A softer Dark Gray is used for body copy.
- Heading styles should use the appropriate font stack for headings, which is designated by the `--font-heading` CSS custom property.
- Heading styles should be used for page titles, section titles, and blockquotes.

### Paragraphs and body copy
- Paragraphs and body copy should use the default font stack.
- Paragraphs and body copy should not exceed 70 characters per line for optimal readability.
- For accessibility, ensure that that there is sufficient contrast between the text and background colors. Visit the **[Colors](/docs/foundation-colors--docs/)** page for more information about the color palette used in the Web Theme.
