# Typography

The Web Theme automatically loads the following fonts. Garamond Premier Pro and Galaxie Polaris are hosted on static.nd.edu. We recommend you use the fonts hosted on static.nd.edu as they're optimized for performance and are hosted on Amazon Cloudfront.

## CSS Custom Properties
The following CSS custom properties are used throughout the Web Theme to designate the appropriate font-stack:

```css
/* Default */
--font-default: GP, "Helvetica Neue", Helvetica, Arial, Verdana, sans-serif;

/* Headings */
--font-heading: Garamond-Pro, "Adobe Garamond", Garamond, Georgia, "Times New Roman", Times, serif;
```
