## Usage
Media Mentions are used to showcase instances where a faculty or staff member has been featured in various media outlets. They help build credibility and highlight positive off-campus exposure.

<ul className="list--todo">
  <li>
    ### Dos
    <ul>
      <li>Ensure the media mention is relevant to the audience</li>
      <li>Include proper citations and links to the original source</li>
      <li>Highlight key quotes or takeaways from the mention</li>
    </ul>
  </li>
  <li>
    ### Don'ts
    <ul>
      <li>Avoid using outdated or irrelevant media mentions</li>
      <li>Don't misrepresent the context of the mention</li>
    </ul>
  </li>
</ul>
