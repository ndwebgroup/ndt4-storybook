## Usage

Dialog components can be used in various scenarios, including:
- Displaying important messages or alerts that require user attention.
- Confirming user actions, such as deletions or submissions.
- Providing additional information without navigating away from the current page.
- Displaying people bios or featured content.

## Modifier Classes
| CSS Class                                 | Description                                                                     |
|-------------------------------------------|---------------------------------------------------------------------------------|
| `.dialog--image`                          | Modifier class for Dialogs that include an image.                               |
| `.dialog--video`                          | Modifier class for Dialogs that include a video.                                |
| `.dialog--person`                         | Modifier class for Dialogs that include a person bio.                           |

## CSS References
| CSS Class                                 | Description                                                                     |
|-------------------------------------------|---------------------------------------------------------------------------------|
| `.dialog`                                 | Container for the Dialog component.                                             |
| `.dialog-heading`                         | Styles the header section of the Dialog, including title and close button.      |
| `.dialog-body`                            | Wrapper for the main content area of the Dialog.                                |
| `.dialog-title`                           | Styles the title text within the Dialog header.                                 |
| `.dialog-footer`                          | Styles the footer section of the Dialog, typically containing action buttons.   |
| `.dialog-link`                            | Class for link that opens the Dialog, including hover and focus states.         | 
| `.dialog-close`                           | Styles the close button in the Dialog header, including hover and focus states. |
