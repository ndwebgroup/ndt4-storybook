## Usage

<ul className="list--todo">

    ### Dos
    <ul>

      <li>Include "Back to top" links for better user experience on long FAQ pages.</li>

## CSS References
| CSS Class                         | Description                                                                                   |
|-----------------------------------|-----------------------------------------------------------------------------------------------|
| `.faq`                             | Main container for the FAQ component.                                                         |
| `.faq-anchors`                     | Anchor list at the top of the FAQ section for quick navigation to questions.                  |
| `.faq-q`                           | Class for each question in the FAQ.                                                           |
| `.faq-a`                           | Class for each answer in the FAQ.                                                             |
