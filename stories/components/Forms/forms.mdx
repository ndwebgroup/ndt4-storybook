## Usage
Forms are essential components for collecting user input on websites and applications. They can be used for a variety of purposes, including data collection, content filtering, and user authentication.

<ul className="list--todo">
  <li>
    ### Dos
    <ul>
      <li>Use clear and concise labels for each form field</li>
      <li>Group related fields together for better organization</li>
      <li>Provide helpful placeholder text to guide users</li>
      <li>Implement validation to ensure data integrity</li>
    </ul>
  </li>
  <li>
    ### Don'ts
    <ul>
      <li>Avoid overwhelming users with too many fields</li>
      <li>Don't forget to provide feedback on form submission</li>
    </ul>

## Wufoo
Wufoo is a popular online form builder that allows users to create and customize forms without any coding knowledge. If you're using Wufoo forms, Notre Dame Creative provides a style sheet to ensure that your forms align with the university's branding guidelines.

**Link:** [https://conductor.nd.edu/stylesheets/wufoo-ndt4.css/](https://conductor.nd.edu/stylesheets/wufoo-ndt4.css/)\
**Usage:** Include in Wufoo's theme designer.
