## Usage

Avatar components can be used to represent users in a variety of contexts, such as article authors, blockquote cites, and profile pictures. They help to personalize the user experience and make it easier to identify users at a glance.

## Accessibility
Avatar components should include appropriate alt text for screen readers. If the avatar represents a user, the alt text should include the user's name. If the avatar is purely decorative, an empty alt attribute (alt="") should be used to prevent screen readers from announcing it.
