Breadcrumbs indicate the current page's location within a navigational hierarchy and help users understand where they are in the website's structure.

<Primary />

<Controls />

---

## Usage

Breadcrumbs should be displayed above the page title on level 2 or lower pages to enhance navigation and provide users with a clear understanding of their location within a website.

## Accessibility
Breadcrumbs should be implemented using a `<nav>` element with an `aria-label` attribute set to "Breadcrumbs". The current page should be indicated using `aria-current="page"` on the last breadcrumb item.

## CSS References
| CSS Class                                 | Description                                                              |
|-------------------------------------------|--------------------------------------------------------------------------|
| `.breadcrumbs`                            | Container for the breadcrumb component.                                  |
| `.breadcrumbs li a::after`                | Adds a separator (e.g., "/") between breadcrumb items.                   |
| `.breadcrumbs li[aria-current="page"]`    | Styles the current page breadcrumb item to indicate it is not a link.    |
