## Usage

The Page Title component can be used as follows:

```jsx
import PageTitle from './PageTitle';

const MyPage = () => {
  return (
    <div>
      <PageTitle title="My Page Title" size="lg" />
    </div>
  );
};
```

 ## Modifier Classes
 | CSS Class                                 | Description                                        |
|--------------------------------------------|----------------------------------------------------|
| `.page-title--xs`                          | Styles the page title with an extra small size.    |
| `.page-title--sm`                          | Styles the page title with a small size.           |
| `.page-title--md`                          | Styles the page title with a medium size.          |
| `.page-title--lg`                          | Styles the page title with a large size.           |
| `.page-title--xl`                          | Styles the page title with an extra large size.    |
