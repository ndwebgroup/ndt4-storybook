## Modifier Classes
| Class Name                        | Description                                                        |
|-----------------------------------|--------------------------------------------------------------------|
| `.gallery--slider`                 | Displays the gallery items in a slider layout.                     |
| `.gallery--tiled`                  | Displays the gallery items in a tiled layout.                      |

## CSS References
| CSS Class                         | Description                                                        |
|-----------------------------------|--------------------------------------------------------------------|
| `.gallery-wrapper`                 | Main container for the Gallery component.                          |
| `.gallery-lb`                      | Class for the individual gallery item.                             |
