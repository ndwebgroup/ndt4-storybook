## Usage
The Stickers component can be used to add visual interest and branding elements to your application. They can be placed in various components to enhance the user interface.

  <li>
    ### Dos
    <ul>

## Modifier Classes
| CSS Class                                 | Description                                                  |
|-------------------------------------------|--------------------------------------------------------------|
| `.sticker--sm`                            | Applies small size to the sticker.                           |
| `.sticker--md`                            | Applies medium size to the sticker.                          |
| `.sticker--lg`                            | Applies large size to the sticker.                           |
| `.sticker--xl`                            | Applies extra large size to the sticker.                     |
