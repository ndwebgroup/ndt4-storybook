## Usage

The Byline component can be used in various contexts, such as articles, blockquotes or media mentions. It helps to provide a consistent and visually appealing way to present author information.

---

## Accessibility

Ensure that the Byline component is accessible by providing appropriate alt text for images and using semantic HTML elements.

---

## CSS References

| CSS Class                                 | Description                                                              |
|-------------------------------------------|--------------------------------------------------------------------------|
| `.byline`                                 | Container for the Byline component.                                      |
| `.byline-body`                            | Wrapper for the text content of the Byline.                              |
| `.byline-image`                           | Styles the author's image, including size and shape.                     |
| `.byline-title`                           | Styles the author's name, including font size and weight.                |
| `.byline-group`                           | Creates a flex container for grouping multiple Byline components.        |
