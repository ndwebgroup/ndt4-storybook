## Modifier Classes
| CSS Class                   | Description                                      |
|-----------------------------|--------------------------------------------------|
| `.stat-item--center`        | Centers the content of the stat item.            |
| `.stat-value--underlined`   | Adds an underline to the stat value.             |
| `.stat-value--xs`           | Applies extra small text size to the stat value. |
| `.stat-value--sm`           | Applies small text size to the stat value.       |
| `.stat-value--md`           | Applies medium text size to the stat value.      |
| `.stat-value--lg`           | Applies large text size to the stat value.       |

## CSS References
| CSS Class                   | Description                                      |
|-----------------------------|--------------------------------------------------|
| `.stat-item`                | Container for each individual stat item.         |
| `.stat-value`               | Styles the value text within the stat item.      |
| `.stat-label`               | Styles the label text within the stat item.      |
| `.stat-sticker`             | Styles the sticker element within the stat item. |
