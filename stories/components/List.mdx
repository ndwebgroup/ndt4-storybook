## Usage
<ul className="list--todo">
  <li>
    ### Dos
    <ul>
      <li>Use an “Unordered List” when you don’t need to convey a specific order for list items</li>
      <li>Use an “Ordered List” when the sequence or count of items is important</li>
      <li>Use a “Description List” with terms and descriptions, such as a glossary, or to display metadata, such as contact details</li>
    </ul>
  </li>
  <li>
    ### Don'ts
    <ul>
      <li>Don't overruse lists for long-form or narrative content.</li>
    </ul>
  </li>
</ul>

## Accessibility

Ensure lists are properly structured using semantic HTML elements (`<ul>`, `<ol>`, `<li>`, `<dl>`, `<dt>`, `<dd>`) to enhance accessibility and readability.

## Modifier Classes
| CSS Class                                 | Description                                                                  |
|-------------------------------------------|------------------------------------------------------------------------------|
| `.list--unstyled`                         | Removes default list styling (bullets or numbers) from the list.             |
| `.list--stepped`                          | Styles the list as a stepped list, often used for processes or instructions. |
| `.list--inline`                           | Displays list items inline, horizontally.                                    |
