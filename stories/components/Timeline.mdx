## Modifier Classes
| CSS Class                                 | Description                                                  |
|-------------------------------------------|--------------------------------------------------------------|
| `.timeline--left`                         | Aligns the timeline to the left.                             |
| `.timeline--right`                        | Aligns the timeline to the right.                            |
| `.timeline--center`                       | Centers the timeline on the page.                            |
