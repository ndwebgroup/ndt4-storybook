## Usage

<ul className="list--todo">
  <li>
    ### Dos
    <ul>
      <li>Use button lists to organize related actions.</li>
      <li>Consider the context and importance of each button when grouping.</li>
    </ul>
  </li>
  <li>
    ### Donts
    <ul>
      <li>Do not mix different types of buttons in the same list.</li>
      <li>Avoid long lists of buttons that can overwhelm users.</li>
    </ul>
  </li>
</ul>

---

## Accessibility

- Ensure all buttons have accessible labels.
- Use semantic HTML elements (e.g., `<a>` and `<button>`) for buttons.
