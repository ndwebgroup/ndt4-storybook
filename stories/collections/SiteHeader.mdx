### CSS References
| CSS Class                                 | Description                                                                    |
|-------------------------------------------|--------------------------------------------------------------------------------|
| `.site-header`                            | Container for the Site Header                                                  |
| `.header-group`                           | Group container for header elements                                            |
| `.header-nav`                             | Container for the navigation elements                                          |
| `.header-util`                            | Container for utility elements such as search icon or cta button               |

### Modifier Classes
| CSS Class                                 | Description                                                                    |
|-------------------------------------------|--------------------------------------------------------------------------------|
| `.header-group--inline-ml`                | Modifier class for inline header elements at 960px                             |
| `.header-group--inline-lg`                | Modifier class for inline header elements at 1280px                            |
| `.header-group--inline--xl`               | Modifier class for inline header elements at 1440px                            |
| `.header-group--inline--xxl`              | Modifier class for inline header elements at 1600px                            |
| `.header-group--logo`                     | Modifier class for swapping title with logo                                    |
