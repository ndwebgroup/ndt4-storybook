## Best Practices 
Designed for maximum visual impact, the <a href="/?path=/story/collections-page-header-page-header-screen--default">Page Header (Screen)</a> collection should be used sparingly and only on the <a href="/?path=/story/templates-home-page--default">Home Page</a> to prevent visual fatigue and maintain its effectiveness.

---

## Usage

<ul className="list--todo">
  <li>
    ### Dos
    <ul>
      <li>Use high-quality images that are relevant to the content of the page header.</li>
      <li>Use the built-in Page Header components to overlay accessible text.</li>
      <li>Ensure the page header is placed at the top of the page.</li>
      <li>Keep the page header concise and focused on the main message or theme of the page.</li>
    </ul>
  </li>
  <li>
    ### Donts
    <ul>
      <li>Avoid using infographics, charts, maps, diagrams, group photos, or detailed portraits.</li>
      <li>Avoid baked-in text, which will pixelate, crop poorly, or get obscured.</li>
      <li>Do not use more than one page header per page.</li>
      <li>Do not overload the page header with too much text.</li>
    </ul>
  </li>
</ul>
